基本用法

xxxxxxxxxx

六种标明行为（i,g,m,y,u,s）

g:表示全局模式(global)，在找到第一个匹配对象后不会直接停止

xxxxxxxxxx
/at/g;

i:表示不区分大小写(case-insensitive)模式，寻找匹配的第一个字符串

xxxxxxxxxx
/[bc]at/i;
//找到第一个bat或者是cat的，不区分大小写

m:表示多行（multiline）模式,找不到会进行下一行的搜索

y:粘附模式，展示只找lastIndex及以后的字符串

u:Unicode模式，开启Unicode匹配

s:dotAll模式，表示元字符.匹配任何字符（包括\n或\r）

正则表达式的元字符：（ [ { \ ^ $ | ) ? * + . ] } ）

转义会需要在这些元字符和其他正则表达式（如\w,\d）的前面加上再多加上一个“\”

exec()方法

即使设置了全局模式（g）,它每次也只会返回一次匹配项。不是全局模式多次调用会返回都是第一个项，全局模式会继续寻找新的匹配对象，原因是全局模式下pattern的lastIndex属性每次都会增加，而非全局模式保持不变。

xxxxxxxxxx
let text="mom and da and bady";
//?是捕获组的
let pattern=/mom(and dad(and bady)?)?/gi;
​
//现在match是一个数组array实例,而现在match现在有两个属性：index和input
//index：匹配项的在字符中的位置
//input: 原正则表达式
let match=pattern.exec(text);
match.index==0 //因为这个字符串本身与这个字符串匹配
​
//上面运行过程：第一项匹配整个字符串 > 在第一项的基础上匹配第二项 > 在第二项的基础上找第三项

迭代器

这些迭代必须使用特殊的Symbol.iterator作为键

具备Iterable接口的内置类型： 1.字符串 2.数组 3.映射 4.集合 5.arguments对象 6.NodeList等DOM集合对象

接受可迭代对象的原生语言特性包括 1.for-of循环 2.数组结构 3.扩展操作符（...） 4.Array.from() 5.创建集合 6.创建映射 7.Promise.all()接受由期约组成的可迭代对象 8.Promise.race()接受由期约组成的可迭代对象 9.yield*操作符（在生成器中使用）

迭代器使用API使用next()方法来遍历数据，每次返回一个IteratorResult对象，它包括两个属性：done和value ，done是boolean值（表示是否在能调用next(),ture表示耗尽）

不同的迭代器之间没有关系。迭代器在迭代期间原数据被修改，迭代器也会相对应的改变

如果迭代器没有关闭，还有继续上从离开的位置开始迭代，比如数组迭代器

        }
    }
}

生成器

生成器也实现了Iterator接口，也具备next()方法 也是返回两个属性done和value

生成器本身就是函数，它可以作为可迭代对象，基本用法

yield关键字只能在生成器内部

提前退出的两种方法return()和throw()

return()方法：提前终止生成器，只要通过它进入关闭的就无法恢复，之后调用next()都是{ value: undefined, done: true }

throw()方法：作用捕获异常，它是需要配合try...catch(e)来抛出异常的，直接在外面用没有什么意义，它不像return()方法直接退出，内部处理了错误还是可以恢复执行的， 内部没有处理的话生成器就会关闭。错误处理会跳过对应的yield

xxxxxxxxxx
function *generator(){
    yield 'foo';
    return yield '11' //(可以用于输入和输出，还有就是提前退出生成器标记为done:true)
}
let gen=generator();
//第一次next()是调用
console.log(gen.next());//{value:'foo',done:false}
//第一次传入值是没有意义的，是先调用
console.log(gen.next("haha"));//{ value: '11', done: false }
//同时输入和输出
console.log(gen.next("haha"));//{ value: 'haha', done: true }
​
//throw
function *generator(){
    for(const x of [1,2,3]){
        try{
            yield x;
        }catch(e){
            console.log(e);
        }
    }

console.log(g.next());//{ value: '1', done: false }
g.throw('foo');//foo
console.log(g.next());//{ value: '3', done: false }
